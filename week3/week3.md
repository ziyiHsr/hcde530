# Week 3 – Coding Studio

This work demonstrates C2: Code Literacy and Documentation. I reorganized a survey analysis script by breaking it into well-named functions and adding docstrings to clarify what each part of the code does. This helped make the overall structure easier to read and modify.

I also added inline comments to explain key steps in the workflow, including how messy input data is cleaned and how results are generated. The script now produces a cleaned CSV file that can be reused for further analysis, making the process more consistent and transparent.
